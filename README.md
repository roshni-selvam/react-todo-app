# ✅ React To-Do List App

A task management app built with **React** for managing daily tasks efficiently.

## Features
- ➕ **Add, Edit & Delete** tasks
- 🔍 **Filter** — All, Completed, Pending
- 💾 **localStorage** persistence
- ⌨️ **Enter key** support
- 📱 **Mobile Responsive** design
- 🎨 **Dark Theme** with purple accent

## Tech Stack
- **React** + **Vite**
- **CSS3** (Flexbox, Media Queries)
- **localStorage API**
- **Poppins Font**

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Build for Production
```bash
npm run build
```

## Project Structure
src/
├── App.js
├── App.css (Mobile Responsive)
├── index.js
└── logo.svg

## Features Explained
- **Add Task:** Type in input field and press Enter or click "Add" button
- **Edit Task:** Click edit icon (✏️) to modify
- **Delete Task:** Click delete icon (🗑️) to remove
- **Filter:** Use dropdown to filter by All/Completed/Pending
