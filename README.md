# ✅ To-Do List App

A task management app built with React and Vite.

## Features
- ➕ Add, Edit & Delete tasks
- 🔍 Filter — All, Completed, Pending
- 💾 localStorage persistence
- ⌨️ Enter key support

## Tech Stack
- React + Vite
- localStorage
- CSS

## Getting Started
```bash
npm install
npm run dev
```
